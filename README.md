# ppp-exercies
This project provides sample solutions for exercises from [Programming â€“ Principles and Practice Using C++ (2nd Edition)](https://www.informit.com/store/programming-principles-and-practice-using-c-plus-plus-9780321992789) (by Bjarne Stroustrup).
They can be built and run with bazel.

```sh
cd ppp-exercies
bazel run -c opt @ppp2e//chapter2:drill
bazel run -c opt @ppp2e//chapter2:ex1
```
